Code:

<button class=" aquabutton standardbutton"> Aqua Button </button>

1. button

The "button" elements states to html that the user is making a new button.

2. class

The class attribute states that the button will follow the "aquabutton", and the "standardbutton" rules.

3. aquabutton standardbutton

The value of attribute tells HTML to show the style "aquabutton", and "standardbutton".

The "aquabutton" can be switched with any HTML supported colors. (bluebutton, graybutton, redbutton)

4. Aqua Button

The text, "Aqua Button" is the test that will be shown inside the button. You can change it with anything you would like.

4. /button

Elements with a "/" with it shows the end of the element statement.
